# TWMailer
A socket-based client-server application in C++ to send and receive messages like an internal mail-server.

## Created by
  - Skyler Mayfield
  - Sohaib Raza Ghulam


### Building
`$ make clean && make`

### Usage
`$ bin/twmailer-server <port> <mail-spool-directoryname>
`$ bin/twmailer-client <ip> <port>
